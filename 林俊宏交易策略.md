## 1. 核心基礎：四大關鍵水平

交易的基礎在於識別圖表上的**四個主要支撐與阻力水平**，用以尋找高機率的交易機會：

- **經典水平 (Classic)**：動態支撐或阻力。當價格觸碰該水平並出現**缺口 (Gap)** 或**未觸及 (Miss)** 而產生延遲回彈，即為高機率設置。
- **突破水平 (Breakout / BO)**：價格以強大力量突破關鍵水平。重點找「乾淨突破」+ 突破缺口（兩邊無干擾）。
- **缺口水平 (Gap)**：同一顏色 K 線之間的空間，最簡單的水平，無需複雜確認。
- **頭肩形水平 (HNS)**：左肩 → 頭部 → 突破後的頸線。需觀察突破、回調與價格拒絕反應。

## 2. 進階過濾工具：EQ 與 CC

用來過濾雜訊，找出最高勝率設置。

### EQ (平衡水平 / Equilibrium)
- **定義**：跨多時間框架、全新未被測試過的支撐/阻力。
- **尋找方式**：從 H4、H1 開始找，再切 M30、M15 確認新鮮度。
- **重要規則**：**只有未測試過**才算 EQ。若已被觸及 → 跳過或改用 CC。EQ 不一定要確認 K 線，搭配趨勢線效果極佳。

### CC (確認 K 線 / Confirmation Candle)
- **定義**：確認市場尊重關鍵水平的特殊 K 線，增加進場信心。
- **核心規則**：CC 方向必須與交易方向一致（多頭看漲 K 線、空頭看跌 K 線）。
- **四種類型**：
  1. **經典 CC**：水平創立或測試後的確認。
  2. **BOCC1**：突破「前」的確認。
  3. **BOCC2**：突破「後」的確認。
  4. **HNS CC**：頭肩形態的確認設置。
- **特例**：BOCC1 與 HNS CC 的確認 K 線方向可能相反（突破前測試）。
- **應用提示**：CC 可在水平剛形成後立即出現，不一定要有 Miss。

## 3. 趨勢線 (Trendline) 的正確畫法與應用

趨勢線在 SNR 系統中是**多功能工具**：減少浮虧、精準進場、過濾無效設置。

### 優質趨勢線繪製標準
- 必須使用 **K 線實體** 畫線，絕對不能畫在影線上。
- 關鍵點距離盡量**對稱**。
- 線條必須**自然**與圖表齊平（不自然寧可不畫）。
- 時間框架必須與支撐/阻力**一致**（M30 水平配 M30 趨勢線）。

### 四種趨勢線類型
1. **延續型 (111)**：三次觸碰都在同一側。
2. **突破型 (11b2 或 1b22)**：包含突破與兩側回測。
3. **HNS 型 (1bb11 或 11bb1)**：與頭肩形態結合。
4. **隱藏 HNS 型 (1b2b1)**：少見但威力強大。

## 4. 進場策略 (Entries)

### 雙重確認進場 (DC - Double Confirmation)
- **方法**：K 線收盤確認後進場。
  - BUY：多頭蠟燭收盤於支撐之上。
  - SELL：空頭蠟燭收盤於阻力之下。
- **優勢**：風險低、穩定，適合新手。
- **最佳時間框架**：M5、M15、H1（M30 勝率較低）。

### 趨勢線進場與反應進場
- **趨勢線進場**：乾淨對稱趨勢線反應時進場，可用小止損（100 點）追求高盈虧比。
- **反應進場**：價格觸碰水平後強烈反應立即進場（切 M1 觀察），適合老手。

### 直接進場 (DE - Direct Entry) 與設置組合
- 熟練後勝率可達 **70%+**。
- **適用框架**：M15、H1 最佳；M5 僅限「無人區」使用。
- **應對法則**：遇到 Setup → 等 DC；遇到 DE → 可直接進場。

### 設置 / DE 組合小抄（高勝率組合矩陣）

| 遇到此水平 \ 碰到此 DE | Gap DE (缺口)       | Breakout DE (突破) | HNS DE (頭肩形)     |
|-----------------------|---------------------|--------------------|---------------------|
| **Gap (缺口)**        | SETUP               | SETUP              | DIRECT ENTRY (DE)   |
| **BOCC (突破CC)**     | SETUP               | DIRECT ENTRY (DE)  | DIRECT ENTRY (DE)   |
| **Classic (經典)**    | SETUP               | DIRECT ENTRY (DE)  | SETUP               |
| **HNS (頭肩形)**      | SETUP               | SETUP              | DIRECT ENTRY (DE)   |
| **Classic CC**        | SETUP               | SETUP              | SETUP               |
| **Breakout (突破)**   | SETUP               | DIRECT ENTRY (DE)  | SETUP               |

## 5. 多時間框架分析與方向判斷

- **故事線方法**：高 TF（M15/H4）看「拒絕」→ 低 TF（M5/H1）找「突破」來確認方向。
- **新價格與突破規則**：專注當下價格，忽略舊價格。突破關鍵支撐並成功拒絕回測 = 新方向成立。
- **注意高階障礙**：絕對不能忽略 H4 阻力，反轉信號可能徹底改變低級別趨勢。

## 6. 風險管理與止損策略

- **提前止損 (Early Stop Loss)**：進場後在 M5 標記翻轉水平。若 M5 K 線實體突破翻轉水平 → **立即手動出場**（保護資金）。
- **目標與盈虧比**：
  - 預設目標 **1R**（穩定收益）。
  - 條件允許可設 **2R / 3R**。
  - 獲利超過 1:2 且無障礙 → 止損移至 **保本 (Breakeven)**，讓利潤奔跑。
